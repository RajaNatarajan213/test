Pre Requistes

1. check the version of the power shell
	$PSVersionTable.PSVersion version should be 5 

2. check if the DEP settings are Turned on.
	control panel -> system and security -> system -> advanced system settings -> performance settings -> data execution prevention -> Turn on DEP for essential Windows Programs and services only.

3. create a group on the local and create two users
	username :- 1. sas
				2. sasdemo

	group :- sasusers

	Administrative Tools -> computer managment -> groups -> create sasusers
	-> user -> create new user ->   1. user name: sas
									2. full name: sas
									3. description: sas insataller user
									4. set password (UNO@1324Pass!)

									1. user name: sasdemo
									2. full name: sasdemp
									3. descripiton: validation user
									4. set password (UNO@1324Pass!)

	add the users to the group sasusers

4. search Local security policy and run it as administrator
	Extend Local Policies -> Right click on User Rights Assignment -> Right click on Log on as a batch job and select properties -> add user or group -> add the group sasusers -> apply

create profile and verify and activate the sas profile(link in mail) download the software. or download it onto the workspace and copy to the sas server

after entering the details the download will start and install
